# Standards

Shared engineering, documentation and repository standards for Aidottami projects.

## Purpose

This repository provides reusable conventions, templates and development tools for:

- `proxmox`
- `HUB-COGNITIVO`
- `AIACT`
- future Aidottami projects

## Contents

- `bash/`: Bash conventions and templates
- `markdown/`: documentation templates and style rules
- `git/`: commit conventions
- `repository/`: repository layout and versioning
- `scripts/`: reusable utilities
- `architecture/`: shared architectural models
- `docs/`: shared terminology

## Version

Current version: `0.1.0`
