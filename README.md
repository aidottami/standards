
# Standards

Shared engineering, documentation and repository standards for Aidottami projects.

## Purpose

This repository provides reusable conventions, templates and development tools for repositories such as:

- `proxmox`

- `HUB-COGNITIVO`

- `AIACT`

- future Aidottami projects

## Contents

- `bash/`: Bash conventions, templates and shared libraries

- `markdown/`: documentation templates and style rules

- `git/`: commit and repository conventions

- `repository/`: layouts, naming and versioning

- `scripts/`: reusable repository utilities

- `architecture/`: provider, operator and agent models

- `docs/`: shared terminology and guidance

## Version

Current version: `0.1.0`
